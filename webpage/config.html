PvME Rotation Overlay

Each profile is one rotation (e.g. a boss/style), split into phases. For each phase, paste the text from PvME's Copy Discord button. In-game, switch profiles with the overlay's dropdown and move between phases with its ◀ / ▶ buttons (or arrow keys while it's focused).
